Visualisations for Addition and Subtraction Problems

You see four dogs at the shelter, Allan, Cook and Murphy and Leo. Allan and Leo get adopted. How many furry friends are left now?

Click the "=" sign to see the results!

HTML

<div class="examples-asvisual-container" id="addSubtDemoVisual">
<!-- add visual here -->
</div>

JS

const subtDiv = document.querySelector( '#addSubtDemoVisual')
const subtVisual = new AddSubVisual([4,2],false,false); subtVisual.makeVisual(subtDiv);
subtVisual.addExplanation("when you have 4 and 2 leave, 2 are left");

CSS

.examples-asvisual-container {
  display: flex;
  justify-content: center;
}
