Getting Started API Examples

API

FractionVisual

Constructor Atrributes

Attribute Type Description
numerator Number/Integer Numerator of the fraction to visualise, corresponds to the number of colored slices in the (pie-chart) visualisation
denominator Number/Integer Denominator of the fraction to visualise, corresponds to the total number of slices in the (pie-chart) visualisation
color String
(accepts hex code, rgb or name)		 Denotes the colour of the filled-in slices of the (pie-chart) visualisation

Methods

/* Generates an instance of the visualisation within html_element */
FractionVisual.makeVisual(html_element)

AddSubtVisual

Constructor Atrributes

Attribute Type Description
numbers Array of 2 Numbers/Integers The numbers depicted as being added(or subtracted) in the visualisation
add Boolean true represents addition and false represents subtraction
choice Boolean The developer can pick between using cat or dog icons for the visualisation -
true represents cats and false represents dogs!

Methods

/* Generates an instance of the visualisation within html_element */
AddSubtVisual.makeVisual(html_element)
/* Add a customised explanation to the visualisation */
AddSubtVisual.addExplanation("count up to four on your fingers!")

MultiplicationVisual

Constructor Atrributes

Attribute Type Description
num1 Number/Integer The first number being multiplied in the visualisation
num2 Number/Integer The second number being multiplied in the visualisation

Methods

/* Generates an instance of the visualisation within html_element */
MultiplicationVisual.makeVisual(html_element)
/* Update the default description to your own customised one */ /* default: "Add up all the groups together!" */
MultiplicationVisual.updateDescription("my new approach to explain multiplication!")

DivisionVisual

Constructor Atrributes

Attribute Type Description
dividend Number/Integer The number to be divided by the divisor in the visualisation
divisor Number/Integer The number dividend is being divided by in the visualisation
uid Number/Integer A unique identifier number to attach to the visualisation for it's elements' IDs.
Tip: We recommend setting it equal to the total number of division visualisations you have on your webpage each time!

Methods

/* Generates an instance of the visualisation within html_element */
DivisionVisual.makeVisual(html_element)

MatrixScalarMultiplicationVisual

Constructor Atrributes

Attribute Type Description
scalar Number/Integer The scalar number being multiplied to the matrix in the visualisation
mat_rows Number/Integer The number of rows in the matrix to be visualised
mat_cols Number/Integer The number of columns in the matrix to be visualised
mat_vals Array of Numbers/Integers values of the elements in the matrix starting from top-left to bottom-right

Methods

/* Generates an instance of the visualisation within html_element */
MatrixScalarMultiplicationVisual.makeVisual(html_element)

MatrixMatrixMultiplicationVisual

Constructor Atrributes

Attribute Type Description
dims1 Array of 2 Numbers/Integers Dimensions of the first matrix being multiplied in the visualisation
Format: [num_rows, num_cols]
dims2 Array of 2 Numbers/Integers Dimensions of the second matrix being multiplied in the visualisation
Format: [num_rows, num_cols]
vals1 Nested Array of Numbers/Integers values of the elements in the first matrix, nested array where each row is an array
Format: [ [row1_col1, row1_col2, row1_col3 ...], [row2_col1, row2_col2, row2_col3 ...], ....]
vals2 Nested Array of Numbers/Integers values of the elements in the second matrix, nested array where each row is an array
Format: [ [row1_col1, row1_col2, row1_col3 ...], [row2_col1, row2_col2, row2_col3 ...], ....]

Methods

/* Generates an instance of the visualisation within html_element */
MatrixMatrixMultiplicationVisual.makeVisual(html_element)
